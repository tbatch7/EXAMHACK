# Exam Hack landing page

Static Vercel-ready landing page for Exam Hack, a triage-first AI study tool.

## Files

- `index.html` - canonical landing page
- `exam-hack-standalone.html` - duplicate standalone preview file
- `colors_and_type.css` - original extracted token file from the first pass

## Deploy on Vercel

This folder can be deployed as a static site. Vercel will serve `index.html` at the root.

## Notes

The current page keeps claims conservative while the product is still being defined. Add real pricing, school logos, usage counts, and performance metrics only once they are true.
